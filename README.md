Project Overview
This project presents an adversary behavior analysis of the threat group APT29 (Cozy Bear) using the MITRE ATT&CK framework. The objective is to understand attacker tactics, techniques, and procedures (TTPs) and identify effective detection strategies from a SOC and threat intelligence perspective.

Objectives
- Analyze a real-world threat actor using MITRE ATT&CK
- Map adversary techniques to attack lifecycle stages
- Understand attacker behavior and detection opportunities
- Develop practical Cyber Threat Intelligence (CTI) skills

Framework Used
- MITRE ATT&CK Framework

Threat Actor Studied
- **APT29 (Cozy Bear)**

 Project Contents
- Threat Actor Profile
- Technique-Level Analysis
- ATT&CK Tactic Mapping
- Detection and Mitigation Recommendations

Skills Demonstrated
- Threat Intelligence Analysis  
- MITRE ATT&CK Mapping  
- Adversary Profiling  
- SOC Detection Thinking  

Future Enhancements
